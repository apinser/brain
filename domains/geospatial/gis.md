# GIS conventions

## Coordinate reference systems
- Swiss reference CRS: **LV95 (EPSG:2056)**
- Exchange/web CRS: **WGS84 (EPSG:4326)**
- Always document the CRS explicitly in project data contracts
- Never assume CRS from file format alone — verify at source

## Evolution
Format standards, tooling conventions (QGIS, GeoPandas, etc.), and output format defaults
are defined at project level until a clear transversal pattern emerges.
